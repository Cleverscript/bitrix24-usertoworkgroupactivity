# Активити бизнес процессов CRM Bitrix24

---

Добавляет блок в дизайнер бизнес процессов.

Что позволяет делать:

1) Добавить пользователя в нужную группу\проект с требуемой ролью
2) Отправить уведомление в чат группы о том что пользователь добавлен
